Tools¶

Prebuilt Tools¶

Tools define graph interactions. Activated tools show up in a toolbar above the graph and only one can be selected at the time.

Currently implemented tools are - SelectCluster - SelectNode - SelectPath - ViewOnly

Read more about how to define your own plugin in Tools.
