KM Configuration¶

Form loader¶

Variables types are range or dropdown.

range takes the form

{
    "type": "range",
    "config": {
        "name": <Variable name>,
        "label": <Display name>,
        "value": <Default value>,
        "max": <Max value>,
        "min": <Min value>,
        "step": <Step>
    }
}

and dropdown takes the form

{
    "type": "dropdown",
    "config": {
        "name": <Variable name>,
        "label": <Display name>,
        "options": <List of options>
    }
}

Current supported Range variables are interval, overlap, dbscan_eps, dbscan_min_samples. For Dropdown type variables, filter_function.

Coloring functions¶

Nodes can be colored in many different ways

{
    "name": "SIZE of node",
    "attr": "size"
}
