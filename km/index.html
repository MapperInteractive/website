We provide off-the-shelf configuration for using Kepler Mapper as the Mapper construction engine. The only customization required is to load the data and pass it to the KeplerMapperConf object.

Following the Getting Started Guide to initialize the basic setup, see the KM Configuration page for more advanced customization, or see Configurations page to learn how to extend the Mapper Interactive framework for yourself.
